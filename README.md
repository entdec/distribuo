# Mobilis
Easy to use in-house distribution of Mobile apps. For now Enterprise and Ad-Hoc apps only.

## Installation
Add this line to your application's Gemfile:

```ruby
gem 'mobilis'
```

And then execute:
```bash
$ bundle
```

Or install it yourself as:
```bash
$ gem install mobilis
```
